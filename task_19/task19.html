Зачастую возникает задача обезопасить данные сайта от копирования. Данная задача очень сложно выполняется на 100%, однако простые способы доработки front end позволяют добиться уже каких-то результатов. Одним из распространенных методов усложнить копирование картинок это просто натянуть поверх другой элемент. Таким образом когда пользователь будет кликать правой кнопкой выши по "картинке" на самом деле клик будет происходить на этом прозрачном элементе. Давайте попробуем это реализовать.


Задачи:
  • Добавьте в тег img атрибут src со значением media/image.jpeg
  • Добавьте в тег с классом image тег div с классом protect
  • Тегу с классом image добавьте относительное позиционирование
  • Тегу с классом protect добавьте абсолютное позиционирование и растените его на всю ширину и высоту родительского элемента
  • Переместите элемент с классом protect в верхний левый угол родительского элемента
